# Smart Freshness Monitoring System

A digital fabrication project focused on food freshness monitoring and waste reduction, aligned with UN SDG 12.

## Project overview
This repository contains two prototype versions:
- Retail / supermarket connected prototype
- Educational / school cafeteria light-box prototype

## Main goals
- Detect relative freshness-related environmental changes around fruit
- Support dynamic discounting in retail settings
- Provide an educational, tangible visualization of food ripening

## Repository structure
- `retail_prototype/` – connected supermarket version
- `educational_prototype/` – autonomous educational version
- `experiments/` – sensor experiments and analysis
- `fabrication/` – laser cutting, 3D printing, and physical iterations
- `docs/` – setup, architecture, and project documentation

## Technologies used
- ESP32 Feather
- ESP32-C3 DevKitM-1
- Adafruit BME688
- Raspberry Pi
- Adafruit ThinkInk 2.13" e-ink display
- NeoPixel Ring 16x RGBW
- Flask
- Arduino IDE

## Status
Prototype in active development for a Digital Fabrication course.
